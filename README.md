# 🌐 Website Clones Collection

A showcase of static website clones developed to master HTML5 and CSS3. This repository features pixel-perfect replications of diverse landing pages and e-commerce sites, demonstrating proficiency in responsive design, layout architectures, and modern UI structuring.

## 📂 Project Directory
This repository contains a central dashboard ([`index.html`](./index.html)) in the root folder that links to all individual projects. Instead of searching through folders, you can easily access them via the dashboard.

| Project Name | Description | Quick Link |
|---|---|---|
| **APEX** | A clone of the APEX website, featuring a dynamic and modern landing page hero section. | [View Project](./APEX/index.html) |
| **ITALIC** | A replication of the Italic luxury goods e-commerce interface, emphasizing minimalist design. | [View Project](./ITALIC/index.html) |
| **Mahindra Auto** | A comprehensive clone of the Mahindra Auto vehicle showcase site. | [View Project](./Mahindra-Auto-Clone/index.html) |
| **NZXT** | A dark-themed custom gaming PC and hardware landing page. | [View Project](./NZXT/index.html) |
| **iCalm** | A calming wellness and stress-relief brand landing page. | [View Project](./icalm/index.html) |

## 🛠️ Tech Stack
* **HTML5:** Semantic markup for accessible and structured content.
* **CSS3:** Custom styling using modern techniques.
* **Flexbox & Grid:** For complex layouts, navigation bars, and product grids.
* **Media Queries:** Ensuring standard responsiveness across web viewports.
* **Animations:** Hover effects and transitions for interactive elements.

## ✨ Key Features
* **Responsive Navigation:** Styled headers and footers for various viewing structures.
* **Hero Sections:** High-quality imagery integrations with overlaid text and CTAs.
* **Asset Management:** Clean, organized structure for images and stylesheets strictly separated per project directory.
* **Vanilla Implementation:** No external styling UI libraries; built from scratch to demonstrate solid fundamental CSS mechanics.

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ammar-Meman/Website-Clone.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Website-Clone
   ```

3. **Open the Dashboard:** 
   Open the root `index.html` file in your preferred web browser. This acts as an interactive table of contents linking to all the individual project clones instantly.

*(Alternatively, navigate directly into specific folders like `/APEX` and open the `index.html` found there.)*
